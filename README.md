<h1 align="left">Automação API Dados Camara</h1>

###

<p align="left">Projeto de automação de teste de API utilizando Postman para consultar, guardar em variáveis e validar os dados dos blocos partidários .<br><br>Nesse projeto é demonstrado duas formas e organizar os testes. A primeira é dividindo as validações de cada bloco em diferentes requests. A segunda é realizando as validações de todos os blocos em uma única request.<br><br>Dados Abertos da Câmara dos Deputados - https://dadosabertos.camara.leg.br/swagger/api.html</p>

###

<h2 align="left">Como Executar</h2>

###

<p align="left">1. Clone este repositório em sua maquina local<br>2. Importe o projeto para o postman<br>3. Selecione o environment "Dados-Camara"<br>4. Rode a collection "Dados-abertos-camara"</p>
